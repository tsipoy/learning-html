Tips on how to write html

How to write html quicker

If you want to write html faster you can follow these instructions.

  1. First, you should respect all the rules in html such as attribute and nesting.If you follow the rules, you may not get stuck writing your code.
  2. Learn a different kind of short cut and apply it when you write your code, which mean use a lot your keyboard instead of using mouse. It is really helpfull if you want to be faster, but it should be in the right way.
  3. Make sure that your code is well written so, when you run it through the html5 validator, you will not spend much time to fix errors or even do it again.
  4. When you start to write your code, it is better to set up your indenting.It will be tiring and just wasting time if fix it after your long html code.
