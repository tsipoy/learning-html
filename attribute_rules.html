Attribute rules

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type.

Attribute values must have quotes

Example
Don't do this
<input type=text name=username>
Do this instead
<input type="text" name="username">

Dush(-) or under score(_) can be used in attribute value

Example
Don't do this
<input type="text" name="artist name"
Do this instead
<input type="text" name="artist-name"
<input type="text" name="artist_name"
