Attribute rules

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.

The attribute rules

  1. Attribute values must have quotes.
  2. Attribute are separated with spaces.
  3. Attribute names need to be lowercase.
  4. An element type has a private attribute.
  5. Public elements can go on any elements for example class or id.
  6. There should not be double quotes inside double quotes.
  7. Dush only can be used in attribut values.

Some examples of attribute rules

Rule number 1 said that attribute values must have quotes
Don't do this
<input type=text name=username>
Do this instead
<input type="text" name="username">

Rule number 7 said that dush only can be used in attribute values
Don't do this
<input type="text" name="artist name"
Do this instead
<input type="text" name="artist-name"
