Nesting rules

Html tag should be nested in a proper order. So, if we wants have a nice code ,we have to the nesting rules.

End the element inside the parent element before closing the parent element.

Don't do this
<p> there is a <b> big problem in this code. </p>
There is an error in this example.

Do this instead
<p> there is a <b>big<b> problem in this code </p>.
We need to close the <b> tag before ending the <p>.

Nest element properly

Don't do this
.....<head>
.....<title>
.....Nesting rules
.....</title>
.....</head>

Do this instead
.....<head>
.......<title>
.........Nesting rules
.......</title>
.....</head>
