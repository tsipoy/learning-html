List of all html rules

To make our code better and useful, we have to follow rules in html.

A list off all html rules

  1. Define a DOCTYPE at the top of the html code.
  2. Can't have an end tag whithout a start tag.
  3. Should use lowercase in elements name.
  4. No <> inside a tag.
  5. Nest element properly.
  6. We should indent child element properly.
  7. Element types have private attribute.
  8. Public attribute can go in any elements for example class or id.
  9. Attribute values must have quotes.
  10. Attribute are separated with spaces.
  11. Attribute names need to be lowercase.
  12. There should not be double quotes inside double quotes.
  13. Dush only can be used in attribut values.
  14. We can't have an end tag whithout a start tag.
  15. Every start tag needs to start with a (<), and end with a (>)
  16. Every end tag needs to start with a (</) and end with a (>)
  17. No (<>) inside the tag.
  18. End the element inside the parent element before ending the parent element.
  19. Nest elements properly.
