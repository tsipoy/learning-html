This website is created for helping people to know more about html.But first, you may ask what is html. Html is an abbreviation of HyperText Markup Language, the code that describe pages and also a software solution . It has already in a fith version. So, in this website we will focus on the html5 which is the latest major version of HyperText Markup Language.

What is html5?
Html5 is the latest evolution of the html, new version of the html, with new elements, attributes, behaviors, and a larger set of technologies that allows the building of more diverse and powerful Web sites and applications.
